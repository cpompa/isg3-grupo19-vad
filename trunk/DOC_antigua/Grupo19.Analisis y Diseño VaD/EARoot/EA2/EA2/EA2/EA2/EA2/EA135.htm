UC-0002 Gestión de Solicitudes/Demanda de Servicios
Versión 1.0 (01-06-2012)
Autores Macarena Rodriguez Rodriguez (Grupo 19)
Fuentes
Objetivos asociados
Requisitos asociados Registro de Voluntarios y Servicios
Descripción Un demandante logado desde su menú de Inicio solicita un servicio.
Una vez rellenado el formulario de solicitud de servicios/actividades el sistema guarda la solicitud. Una vez guardada la solicitud, el sistema busca si existe algún usuario que cumple con los requisitos solicitados por el demandante. Si es así informa al usuario que se ha mandado la solicitud al voluntario en caso contrario informa de que no existe ningún voluntario que ofrezca los servicios solicitados. Si el voluntario existe el sistema registra la solicitud en la bandeja de solicitudes del voluntario.

Precondición El usuario está registrado como demandante previamente.

Secuencia normal Paso Acción
p1 El usuario se loga en la aplicación y está registrado como demandante.
p2 El sistema muestra el menú de inicio del demandante con la opción “Solicitud de Servicios”
p3 El sistema solicita los campos obligatorios para la creación de una solicitud de servicios
p4 El usuario rellena los datos y los envía
p5 El sistema registra los datos de la solicitud.
P6 El sistema busca si existe algún usuario que cumpla con los requisitos solicitados por el demandante e informa al usuario por pantalla

P7 Si esta búsqueda es positiva el sistema da de alta la solicitud en la bandeja de solicitudes del voluntario.

Post-condición La solicitud de servicio es dada de alta y si existe un voluntario que ofrezca dicha demanda se le informa en su bandeja de solicitudes.
Mejoras Paso Acción
P5 Mandar un correo al usuario voluntario y al administrador con la solicitud de una demanda de sus servicios.
p5 Mandar un correo al administrador del sistema con una demanda de solicitud no satisfecha
