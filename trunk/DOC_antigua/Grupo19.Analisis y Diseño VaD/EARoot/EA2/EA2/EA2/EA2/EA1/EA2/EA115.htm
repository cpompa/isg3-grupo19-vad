DESCRIPCIÓN:
Mantenimiento de perfiles.
Los campos mostrados en el filtro seran código y descripción.



PRECONDICIÓN:


SECUENCIA NORMAL:
Se ejecutan el CU-AD0030 con las tablas de PERFILES, USUARIOS-PERFILES.


SECUENCIAS ALTERNATIVAS:
El perfil ya existe

POSTCONDICIÓN:


REFERENCIAS:
ERD-AD0010



PLAN DE PRUEBAS:
* Crear un perfil nuevo sin opciones y comprobar que se guarda correctamente en la bb dd.
* Asignar este perfil a un usuario y comprobar que no le aparecen nuevas opciones de menú.
* Modificar el perfil asignándole varias opciones y comportamientos. Comprobar que si se autentica con el usuario anterior si le aparecen las nuevas opciones y los comportamientos asociados.
* Eliminar el perfil y comprobar que se eliminan tanto el perfil como su relación con opciones como su relación con el usuario en la bb dd.
* Autenticar con el usuario y comprobar que ha perdido las opciones.
