Actores Usuario Admin
Propósito Realizar la consulta de un perfil
Resumen El Usuario solicita la consulta de los datos de un perfil, mostrándosele la ventana con la información del registro.
Tipo Primario y Esencial
Referencias Cruzadas Casos de uso: el usuario debe haber terminado el caso de uso Acceso al Sistema (UC-26).
Requisitos Asociados
Curso normal de los eventos
Acciones de los actores Respuesta del Sistema
1. El Usuario accede a la ventana de mantenimiento de perfiles.
2. Muestra la ventana del mantenimiento de perfiles con todos los registros almacenados, y sus funcionalidades y acciones relacionadas.
Cursos alternos
Línea 1: El Usuario no tiene permisos para realizar la consulta de perfiles. El Sistema muestra mensaje informando y cancela el proceso.
