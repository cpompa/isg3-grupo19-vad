DESCRIPCIÓN:
Mantenimiento de usuarios y asignación de los perfiles que le corresponde. No incluido en esta versión VaD_v1.0.



PRECONDICIÓN:


SECUENCIA NORMAL:
Se ejecutan el CU-AD0030 con las tablas de USUARIOS y PERFILES.


SECUENCIAS ALTERNATIVAS:
El usuario ya existe

POSTCONDICIÓN:


REFERENCIAS:
ERD-AD0010




PLAN DE PRUEBAS:
* Crear un usuario nuevo sin perfiles y comprobar que se autentica, pero no le aparecen opciones
* Modificar el usuario asignándole un perfil y una nueva clave. Comprobar que se autentica y le aparecen las opciones adecuadas según el perfil
* Eliminar el usuario y comprobar que no se puede autenticar.
