Objetivos asociados
Requisitos asociados Registro de Usuarios y Servicios
Descripción Los usuarios deben de registrarse previamente para formar parte de la red social de VaD. Los usuarios pueden registrarse como voluntarios o como demandantes que solicitan servicios. Para ello tienen que dar de alta sus datos personales y después los servicios en los que se van a dar de alta como voluntarios o como demandantes. Esto significa dar de alta al usuario, al perfil y los servicios correspondientes.

Precondición El usuario no está registrado previamente.

Secuencia normal Paso Acción
p1 El usuario accede a la aplicación y no está registrado previamente.
p2 El sistema solicita los campos de los datos personales obligatorios para el registro
p3 El usuario rellena los datos personales y los envía.
p4 El sistema comprueba que el usuario no existe previamente.
p5 El sistema guarda los datos personales en la tabla de usuarios.

p6 El sistema da de alta al usuario en la tabla de usuarios_perfiles

p7 El sistema muestra una pantalla con los datos personales guardados y muestra una lista con los servicios actuales en los que se puede dar de alta
p8 El sistema registra los datos e informa de que los datos han sido registrados.


Post-condición El usuario es registrado en el sistema como voluntario o demandante y han sido dado de alta los servicios que quiere dar o recibir.
Excepciones Paso Acción
p1 Si el usuario ya está dado de alta recibe un mensaje de que ya está registrado y lo manda a la pantalla inicial donde podrá logarse.
